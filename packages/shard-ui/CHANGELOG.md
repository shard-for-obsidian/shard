# @shard-for-obsidian/ui Changelog

## 0.1.0 - 2026-02-06

### Added

- Initial release of shared UI package
- Button component with variants (default, secondary, outline, ghost, link, destructive)
- Card component suite (Root, Header, Title, Description, Content, Footer)
- PluginCard component for displaying marketplace plugins
- SearchBar component with search icon
- Obsidian CSS variable integration via Tailwind bridge
- TypeScript type definitions for all components
- MarketplacePlugin interface
- cn() utility for class name merging

### Infrastructure

- Vite build configuration
- TypeScript with declaration generation
- Svelte 5 with runes mode
- Tailwind CSS 4 integration
- pnpm workspace package
